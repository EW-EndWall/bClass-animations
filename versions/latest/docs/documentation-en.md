# bClass Animations

`bClass animations` is a CSS and JavaScript library that provides ready-to-use animations for web projects. It offers smooth transitions, hover effects, and various animation types to enhance user experience.

## Features

- Smooth CSS animations
- Hover effects
- Transition effects
- Scroll animations
- Opacity animations
- Rotation animations
- Element interaction animations

## Usage

### CSS Classes

#### Animation Types

- `.animate-up` - Animation that moves elements upward
- `.animate-left` - Animation that moves elements to the left
- `.animate-down` - Animation that moves elements downward
- `.animate-right` - Animation that moves elements to the right
- `.animate-slide-right` - Animation that slides elements from right to left
- `.animate-slide-left` - Animation that slides elements from left to right
- `.animate-slide-up` - Animation that slides elements from bottom to top
- `.animate-slide-down` - Animation that slides elements from top to bottom
- `.animate-opacity-sharp-full` - Sharp opacity animation (from 0 to 1)
- `.animate-opacity-sharp-half` - Sharp opacity animation (from 0.5 to 1)
- `.animate-opacity-soft-full` - Soft opacity animation (0 → 1 → 0)
- `.animate-opacity-soft-half` - Soft opacity animation (0.5 → 1 → 0.5)
- `.animate-rotate-l` - Rotates to the left
- `.animate-rotate-r` - Rotates to the right
- `.animate-hover-up` - Animation that moves elements upward on hover
- `.animate-hover-left` - Animation that moves elements to the left on hover
- `.animate-hover-down` - Animation that moves elements downward on hover
- `.animate-hover-right` - Animation that moves elements to the right on hover
- `.animate-items-m-1`, `.animate-items-hovered-m-1` - Style for hovered items
- `.animate-items-m-1`, `.animate-items-not-hovered-m-1` - Style for non-hovered items
- `.animate-right infinite hover:animate-none` - Moves elements to the right in an infinite loop and stops the animation upon hovering

#### Transition Effects

- `transition-none` - No transition
- `transition-all` - All transitions
- `transition-color` - Color transitions
- `transition-opacity` - Opacity transitions
- `transition-shadow` - Shadow transitions
- `transition-transform` - Transform transitions
- `transition-margin` - Margin transitions

#### Animation Timing Functions

- `.linear` - Linear timing function
- `.ease` - Ease timing function
- `.ease-in` - Ease-in timing function
- `.ease-out` - Ease-out timing function
- `.ease-in-out` - Ease-in-out timing function
- `.steps-6-end` - Timing function with 6 steps
- `.cubic-bezier` - Cubic bezier timing function
