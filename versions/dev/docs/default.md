# Default

## No animations

### example

Pages animations off or element.

- .hover:animate-none
- .animate-none

```html
<body class="animate-none">
  content
</body>
```

## Animations paused

Animations hover is paused.

- .animate-paused
- .hover:animate-paused

### example

Pages animations off.

```html
<div class="animations-paused">content</div>
```
